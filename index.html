Consulta Trecho x Modalidade

Utilize esta funcionalidade para consultar os custos e distâncias entre todas as cidades disponíves para transporte.

Cadastrar Transporte

  1. Adicione a origem e os destinos desejados na interface.
    Obs.: É necessário adicionar ao menos uma cidade de origem e uma cidade de destino.
  2. Clique no botão 'inserir' para adicionar itens a serem transportados.
    Será aberta uma janela para adicinar a descrição, peso e quantidade do mesmo.
    Obs.: É necessário adicionar ao menos um item.
  3. Clique em 'gerar' para visualizar o resumo do transporte configurado.
  4. Configure os itens que deseja transportar para as respectivas cidades.
    Obs.: É necessário que a quantidade parcial de itens seja a mesma que a quantidade total.
  5. Clique em 'Cadastrar Transporte' para visualizar as informações do trajeto configurado.
  6. Clique em 'Salvar' para salvar os dados do trajeto.
